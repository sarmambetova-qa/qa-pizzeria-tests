# Проект: автоматизация тестирования интернет-магазина пиццерии

## Технологии
- Python, pytest, Selenium, Playwright
- Page Object Model, Allure, логирование
- SQL (PostgreSQL, DBeaver)
- Docker, Selenoid
- Git, GitLab, YouGile, Postman

## Что тестировалось
- Основной флоу клиента (выбор пиццы, корзина, регистрация, оформление)
- Промокоды (валидный, невалидный, повторное применение)
- Бонусная программа
- Перехват сетевого запроса (Playwright)

## Как запустить тесты
```bash
pytest final_work/tests/
